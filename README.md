# Pyomo Energy System Model

Modular research-style project structure for a small electricity system planning model built with Pyomo.

**Scope**
- Technologies: Coal, Gas, Wind, Solar, Hydro
- Planning years: 2025–2034
- Focus: structure only, model implementation to be added

**Project Layout**
- data/
- model/
- scenarios/
- utils/
- run_model.py
- requirements.txt

**Data Inputs**
- data/technology_params.json
- data/demand_projection.csv

**Usage**
- Create a virtual environment
- Install dependencies from requirements.txt
- Implement model components in model/
- Run run_model.py when ready
